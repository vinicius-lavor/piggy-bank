## Instalação

Para instalar o projeto:

```bash
  git clone https://github.com/vinicius-lavor/piggy-bank.git
```
Para entrar na pasta do projeto:
```bash
  cd pasta-onde-o-projeto-foi-instalado
  cd piggy-bank/Piggy Bank
```
Para executar:
```bash
  open Piggy\ Bank.xcodeproj                                                                                                                                
```
